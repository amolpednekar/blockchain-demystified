# Web3 Setup

## Install NodeJS:

* [Install NVM (Node version manager)](https://github.com/creationix/nvm)
* Install NodeJS
```
nvm install 8.0.0

#Check node and npm  version
node -v
npm -v
```

## Install web3
```
npm install --save web3
```
* [Web3 Docs](https://github.com/ethereum/wiki/wiki/JavaScript-API#web3ethaccounts)
